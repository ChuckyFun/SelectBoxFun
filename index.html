JavaScript

                    
// Bu şartı sağlayan tüm selectleri etkiler
$('.custom_select select').CFSelectBox();

// Sadece bu şartı sağlayan selecti etkiler
$('.custom_select select').CFSelectBox({
    type: 'single'
})

// perfectScrollbar jQuery plugin ile kullanımı
$('.custom_select select').CFSelectBox({
    usePerfectScroll: true,
    type: 'single'
});

Kullanabileceğiniz Parametreler

scrollItemLimit (Varsayılan: 9)

Bu parametre ile scroll gösterimi için bir limit belirtebilirsiniz. Varsayılan değer 9'dur ve 9'dan fazla seçenek olduğunda scroll yapılacağını gösterir.

scrollMaxHeight (Varsayılan: 180)

Bu parametre ile scroll gösterimi için bir yükseklik limiti belirtebilirsiniz.

usePerfectScroll (Varsayılan: false)

Bu parametre ile scroll gösterimi tarayıcının varsayılan görüntüsü olarak kullanmak istemiyorsanız ve perfectScrollbar jquery plugin yüklüyse true değeri verebilirsiniz.

type (Varsayılan: 'all')

Bu parametre ile çağırılan selectlerin hepsini veya sadece birini etkilemesi için kullanabilirsiniz.
